# Basic ChatBot
- A simple project that responds to user queries with some predefined responses.

## Features
- User input
- Responses to greetings and some basic questions.
- Handle unknown inputs
- Simple command line interface
- Allows user to exit the chatbot

## Technologies used
- If-Elif statements
- Strings
- Input/Output
- Loop

## Example interaction
- You : Hi
- Bot : Hello! How are you?
- You : What is your name
- Bot : My name is CodeAlpha Bot.
